# Zomato Customer Segmentation Analysis

## Project Overview
This project analyzes Zomato's customer data to identify user segments, purchasing behaviors, and preferences. As a junior analyst for Zomato, the objective is to provide insights that inform marketing strategies and improve user engagement on the platform.

## Dataset
The dataset for this project is provided in a zip file named **Zomato data.zip**, which contains the following five tables:
- **food**: Details about various food items available for order.
- **menu**: Information on the menus of different restaurants.
- **orders**: Transaction records detailing customer orders.
- **restaurant**: Information about the restaurants listed on Zomato.
- **users**: User demographic information.

These tables serve as the foundation for comprehensive analyses focusing on customer segmentation and purchasing behavior.

## Introduction
Zomato is a multinational restaurant aggregator and food delivery company. In the face of growing competition, understanding the customer base remains crucial for formulating effective marketing strategies. The analysis focuses on customer segmentation based on demographics, ordering behavior, and preferences to identify opportunities for enhancing customer engagement and retention.

## Project Objectives
- Segment Zomato's users based on demographics, order frequency, and spending patterns.
- Identify popular restaurants and cuisines preferred by different user segments.
- Provide actionable recommendations for marketing strategies tailored to specific customer groups.

## Methodology
1. **Data Preparation**: 
   - Import and clean the datasets to ensure accurate analysis.
   - Merge tables as necessary to create a comprehensive view of user behavior and preferences.
  
2. **Exploratory Data Analysis (EDA)**:
   - Conduct descriptive statistics to understand the demographic distribution of users.
   - Analyze user segmentation based on age, gender, marital status, income, occupation, and family size.

3. **Segmentation Analysis**:
   - Identify key segments of users and analyze their purchasing behavior.
   - Create visual representations of the findings using data visualization tools like Tableau.

4. **Reporting**:
   - Prepare a detailed report summarizing the findings and insights.
   - Present a dashboard for interactive exploration of user demographics and behavior.

## Key Findings
1. **User Demographics**:
   - The majority of users are male, aged between 22 and 25, with over 50% being single.
   - Most users report no income, while a smaller percentage earn less than Rs. 10,000.
   - The most common occupations among users are students and employees, with students leading in order frequency and total spending.

2. **Culinary Preferences**:
   - North Indian and Chinese cuisines stand out as the most popular across all demographic segments.
   - A consistent pattern in cuisine preference indicates a need for Zomato to maintain diverse offerings.

3. **Spending Behavior**:
   - Users aged 22 to 26 exhibit the highest engagement levels.
   - Married users show lower order frequency compared to singles, suggesting targeted marketing opportunities.

## Conclusion
The analysis provides valuable insights into Zomato's user base, revealing significant opportunities for targeted marketing strategies. By understanding the demographics and preferences of different user segments, Zomato enhances user engagement, increases order frequency, and improves customer satisfaction.

## Recommendations for Improvement
- **Targeted Marketing Campaigns**: Develop campaigns aimed at engaging underrepresented demographics, such as older users and married individuals.
- **Incentives for Feedback**: Implement promotional strategies to encourage user ratings and feedback, which can help refine service offerings.
- **Partnerships with Restaurants**: Foster partnerships with popular North Indian and Chinese restaurants to offer exclusive deals and promotions.

## Future Directions
- Conduct further analysis into customer feedback and satisfaction to identify specific areas for improvement.
- Explore additional datasets to gain deeper insights into customer preferences and behaviors.
- Implement A/B testing for marketing strategies to evaluate their effectiveness on different user segments.
